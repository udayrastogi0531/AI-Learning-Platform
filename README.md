## 🌟 Overview
Transform the educational experience with this full-stack AI-driven Learning Management System (LMS). Built with cutting-edge technologies like Next.js, Firebase, and Gemini AI, it offers personalized tutoring, role-based dashboards, and a gamified learning journey.

---

## ✨ Key Features
- 🤖 **Gemini AI Integration:** Smart, context-aware assistant for real-time problem solving.
- 👥 **Multi-Role Dashboards:** Dedicated experiences for Learners, Instructors, and Admin.
- 🎥 **Interactive Video Hub:** High-performance video courses with automated progress tracking.
- 🏆 **Gamified Learning:** Achievement badges, certificates, and seamless progress tracking.
- 📱 **Progressive Web App (PWA):** Offline support and push notifications (FCM).
- 💳 **Stripe Monetization:** Ready-to-use course enrollment and secure payment pipeline.

## 🚀 Getting Started

### Prerequisites
- Node.js 18.x or later
- Firebase Project setup
- Gemini API Key

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/udayrastogi0531/AI-Learning-Platform.git
   cd AI-Learning-Platform
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up Environment Variables**
   Create a `.env.local` file and add your keys.

4. **Run the development server**
   ```bash
   npm run dev
   ```

## ⚙️ Tech Stack & Architecture
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS + UI Components
- **Backend & Auth:** Firebase Services (Auth, Firestore, Storage, Cloud Messaging)
- **AI Intelligence:** Google Gemini Pro
- **Payments:** Stripe API
